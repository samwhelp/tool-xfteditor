
# Xftvi


## Target Editor

* vi


## Install

``` sh
make install
```

* [xftvi](xftvi)
* [xftvi.desktop](xftvi.desktop)

## Remove

``` sh
make remove
```

## Usage

### On File Manager

1. Launch FileManager. For example: `thunar`
2. Specific File
3. Open With Xftvi

### On xfce4-terminal

1. Launch `xfce4-terminal`
2. Run `xftvi file`
