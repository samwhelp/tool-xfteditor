
# Xftnvim


## Target Editor

* [Neovim](https://neovim.io/)


## Install

``` sh
make install
```

* [xftnvim](xftnvim)
* [xftnvim.desktop](xftnvim.desktop)

## Remove

``` sh
make remove
```

## Usage

### On File Manager

1. Launch FileManager. For example: `thunar`
2. Specific File
3. Open With Xftnvim

### On xfce4-terminal

1. Launch `xfce4-terminal`
2. Run `xftnvim file`
